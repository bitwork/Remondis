Es handelt sich um JS Code der in der Editform und Newform.aspx bei Remondis verknüpft wird.
Die htm Datei wird in den SiteAssets gespeichert und von dort referenziert.

Der Code prüft auf das Genehmigerfeld und vergleicht den eingetragenen Wert mit dem aktuell angemeldeten Benutzer.
Wenn diese übereinstimmen wird eine Fehlermeldung angezeigt
